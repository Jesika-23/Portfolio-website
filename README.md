# Accessible Personal Portfolio Website

## 📌 Project Overview
This is a multi-page personal portfolio website built using **HTML5 semantic structure** and **basic CSS styling**.  
It follows **web accessibility (WCAG) guidelines** and includes SEO-friendly meta tags.

The project is designed as a beginner-friendly portfolio to showcase skills, projects, and contact information.

---

## Features

- Multi-page structure (Home, About, Projects, Contact)
- Semantic HTML5 tags (`header`, `nav`, `main`, `section`, `article`, `footer`)
- Accessible contact form with proper labels
- SEO-friendly meta tags
- Simple and clean CSS styling
- Keyboard-navigable layout
- Fully responsive-ready structure (basic level)

2. Open the `portfolio` folder
3. Open `index.html` in any web browser
